Slither
=======

To execute the program:

1. Navigate to directory containing program files
2. Execute make
3. Run ./main
4. Input B1.txt
5. Enter C
6. Wait for solution
7. Enter Y
8. Input B2.txt
9. Enter C
10.Wait for solution


Data File Format:
See B1.txt for an example of a 5 x 5 grid.
The 7s represent blank squares with no value.
